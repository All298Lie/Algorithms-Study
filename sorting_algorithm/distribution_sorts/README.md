# 계수 정렬 (Counting Sort)

## 1. 개요 및 복잡도
### 시간 복잡도
최선 : $O(N+K)$ (데이터의 개수가 $N$, 데이터의 최댓값이 $K$일 때 항상 동일한 연산 횟수를 가짐)  

평균 : $O(N+K)$  

최악 : $O(N+K)$ (데이터의 초기 분포 상태에 영향을 받지 않음)  

### 공간 복잡도
$O(N+K)$ (데이터의 최댓값 $K$ 크기를 가지는 카운팅 배열과 $N$ 크기의 결과 배열이 추가로 필요함)  

### 핵심 요약
- 원소들 간의 크기를 직접 비교하지 않고, 각 데이터가 몇 번 등장했는지 개수를 세어 위치를 결정하는 방식  
- 데이터의 크기 범위가 제한된 정수 형태일 때, 비교 정렬의 이론적 하한선인 $O(N\log N)$을 뛰어넘어 선형 시간($O(N)$)에 가까운 압도적인 속도를 자랑하는 알고리즘

## 2. 동작 원리
1. 정렬할 원본 배열에서 최댓값($K$)을 찾아, $0$부터 $K$까지의 인덱스를 갖는 ‘카운팅 배열’을 생성하고 모두 $0$으로 초기화한다.  
2. 원본 배열을 순회하며, 각 원소의 값을 인덱스로 삼아 카운팅 배열의 값을 $1$씩 증가시킨다.(데이터의 등장 빈도수 측정)  
3. 빈도수 측정이 끝나면, 카운팅 배열의 값들을 처음부터 끝까지 누적합으로 갱신한다. (각 데이터가 최종 결과 배열에서 들어갈 마지막 위치를 계산하는 과정)  
4. 원본 배열을 맨 뒤에서부터 맨 앞으로 역순으로 순회하며 정렬을 시작한다.  
5. 원본 배열의 값을 카운팅 배열의 인덱스로 찾아가, 기록된 누적합 값을 $1$ 감소시키고, 그 감소된 값을 새로운 결과 배열의 인덱스로 사용하여 원소를 배치한다. (역순 순회를 통해 안정 정렬을 보장)  

## 3. 장단점 및 특징
### 장점
- 데이터의 크기 범위($K$)가 작고 촘촘하게 모여 있을 경우, 어떠한 비교 연산도 수행하지 않으므로 실제 동작 속도가 퀵 정렬보다도 훨씬 빠르다.  
- 안정 정렬 특성을 가져, 값이 같은 원소들의 입력된 순서가 완벽하게 유지된다.  

### 단점
- 데이터의 최댓값이 매우 크면 해당 크기의 배열을 만들어야 하므로 메모리 낭비가 극심해지고 초기화에 엄청난 시간이 소모된다.  
- 배열의 인덱스를 활용하는 방식이므로, 소수점이 있는 실수 데이터나 음수가 포함된 데이터는 위치를 보정하는 추가 작업 없이는 정렬할 수 없다.  

### 특이사항
- 안정 정렬 : 값이 같은 데이터의 초기 순서가 정렬 후에도 그대로 유지된다. (단, 결과 배열을 채울 때 반드시 원본 배열을 역순으로 탐색해야 한다.)  
- Out-of-place Sort : 제자리 정렬이 아니며, 카운팅 배열과 결과 배열이라는 두 개의 별도 메모리 공간이 요구된다.  

## 4. 구현 (C++)
[CountingSort.cpp 전체 코드 파일 확인](./CountingSort.cpp)
```cpp
#include <vector>
#include <algorithm> // max_element 함수 사용

using namespace std;

// 계수 정렬 메인 함수
void CountingSort(vector<int>& arr) {
    if (arr.empty()) return;

    int n = arr.size();

    // 1. 원본 배열에서 최댓값 찾기
    int max_val = *max_element(arr.begin(), arr.end());

    // 2. 카운팅 배열(Counting Array) 생성 및 0으로 초기화
    // 인덱스가 최댓값까지 있어야 하므로 크기는 max_val + 1
    vector<int> count(max_val + 1, 0);

    // 정렬된 결과를 임시로 담을 배열
    vector<int> output(n);

    // 3. 각 원소의 등장 횟수 세기
    for (int i = 0; i < n; i++) {
        count[arr[i]]++;
    }

    // 4. 카운팅 배열을 누적합(Prefix Sum)으로 변환
    // 각 값이 결과 배열에서 위치할 마지막 인덱스를 계산
    for (int i = 1; i <= max_val; i++) {
        count[i] += count[i - 1];
    }

    // 5. 원본 배열을 뒤에서부터 순회하며 결과 배열에 배치 (안정 정렬 보장)
    for (int i = n - 1; i >= 0; i--) {
        int val = arr[i];
        
        // 누적합에서 1을 뺀 값이 결과 배열에서의 실제 인덱스
        count[val]--;
        output[count[val]] = val;
    }

    // 6. 정렬된 결과를 원본 배열에 덮어쓰기
    for (int i = 0; i < n; i++) {
        arr[i] = output[i];
    }
}
```
